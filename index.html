Steps Implemented

01

Choose a game type

Two distinct game modes: a number-guessing game with three difficulty levels (Easy / Medium / Hard) and a multiple-choice general knowledge quiz.

02

Define game rules and logic

Guessing game: pick a random secret, award points based on fewest attempts. Quiz: 5 curated questions with A–D choices and instant feedback.

03

Conditional statements to manage game flow

Every branch uses if / elif / else — menu routing, difficulty selection, guess comparison (too low / too high / correct), quiz answer checking, and replay prompts.

04

Test and debug for correctness

Input validation loops reject non-numeric entries and out-of-range values. Edge cases (boundary numbers, empty input) are all handled gracefully.

# Step 3 — conditional statements managing game flow

if guess < secret:
    print("  📈  Too low!  Try higher.")

elif guess > secret:
    print("  📉  Too high! Try lower.")

else:                             # guess == secret
    print(f"🎉  Correct in {attempts_used} attempt(s)!")
    return True, attempts_used
